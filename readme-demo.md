# SimpliFy Demo Guide

This guide provides instructions on how to demo the different features of the SimpliFy extension.

## Autism & Cognitive Support

This module is designed to help users with cognitive disabilities by simplifying web content.

**Test URL:** [https://sites.google.com/view/simplifydemo/complex-article?authuser=0](https://sites.google.com/view/simplifydemo/complex-article?authuser=0)

### Features to Demo

1.  **Text Simplification:**
    *   Navigate to the provided test URL.
    *   Open the SimpliFy extension and go to the "Autism & Cognitive Support" module.
    *   Click the "Simplify" button. The complex text on the page will be replaced with a simplified version.

2.  **Upload Audio for Simplification:**
    *   From the same module, click the "Upload" button.
    *   Select an audio file to receive a simplified text version of the audio content.

3.  **Toggle Animations:**
    *   Click the "Toggle" button to turn off animations on the page, reducing distractions.

### Settings

Explore the "Settings" tab within the module to customize the experience:

*   **Simplification Level:** Adjust the intensity of the text simplification.
*   **Toggle Animations:** A switch to control animations.
*   **Read-Aloud Speed:** Control the speed of the text-to-speech feature.
*   **Volume:** Adjust the volume of the audio output.
*   **Content Filtering:** Enable or disable filtering of adult/profane words.

## Auditory Accessibility

This module provides tools for users who are deaf or hard of hearing.

**Test URL:** [https://sites.google.com/view/simplifydemo/audio-embedded-article?authuser=0](https://sites.google.com/view/simplifydemo/audio-embedded-article?authuser=0)

### Features to Demo

1.  **Live Audio Transcribe:**
    *   Open the SimpliFy extension and navigate to the "Auditory Accessibility" module.
    *   Click "Start Transcribing" to get real-time transcriptions of any audio playing from your device or spoken into your microphone.

2.  **Transcribe Media Files:**
    *   Navigate to the provided test URL, which contains an embedded audio file.
    *   In the Auditory module, click "Select File" and choose an audio or video file from your computer to transcribe.

3.  **Scribe:**
    *   Go to the "Scribe" tab to view the full transcription of any audio.
    *   You can use the "Download Transcript" button to save the transcription as a text file.

### Settings

Customize the auditory features in the "Settings" tab:

*   **Auto download Transcribe Text:** Automatically save the transcript when you stop recording.
*   **Show timestamps in Transcribe:** Toggle timestamps on and off in the Scribe view.
*   **Filter out Adult/profane words:** Enable or disable the profanity filter.
